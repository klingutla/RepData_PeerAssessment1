This document presents an analysis of the data collected from a larger number of personal activity monitoring devices. The devices collect data at 5 minute intervals through out the day. The data consists of two months of data from anonymous individuals collected during the months of October and November, 2012 and include the number of steps taken in 5 minute intervals each day.

Replacing NA with mean of average steps across all days for that interval

for(i in 1:nrow(activityData)) {
  row <- activityData[i,]
  # do stuff with row
  if (is.na(row[1])) {
    CleanedActivityData[i,1] <- as.numeric(avgSteps[which(avgSteps$Group.1 == row$interval), 2])
  } else {
    CleanedActivityData[i,1] <- as.numeric(row$steps)
  }
  CleanedActivityData[i,2] <- row$date
  CleanedActivityData[i,3] <- row$interval
}

totalStepsInCleanedData <- aggregate(CleanedActivityData$steps, by=list(CleanedActivityData$date), FUN=sum, na.rm = TRUE)
